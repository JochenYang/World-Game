# 寰宇溯源 · 全球人类起源史

中国风 3D 交互展示 — 基于 WebGPU 渲染的水墨地球，覆盖七大洲起源、二十六国列国史、地球生命编年。

## 特性

- **WebGPU 优先渲染** + WebGL 自动回退，水墨质感的地球与大气层 Fresnel 光晕
- **中国风视觉**：朱红、鸦青、缃色、月白、宣纸色板；卷轴、印章、毛笔字、缥缈过渡
- **三层内容架构**：
  - 大陆溯源（8 区，含北极）：每区 14-19 个关键史前节点
  - 地球编年：103 条全球时间轴，从冥古宙到 2024
  - 列国史：26 个文明国家的深度年表（中、印、波斯、希腊、罗马、英、法、俄、埃、马里、印加等）
- **历史生物长卷背景**：21 张古生物卡片（冥古宙到现代城市），自动从右向左循环
- **交互**：拖拽旋转、滚轮缩放、悬停高亮、点击大陆聚焦并展开卷轴
- **地球默认自转**（顺时针）；用户操作时暂停，2.5 秒后恢复
- **音乐播放控制**（MusicManager + MusicToggle）

## 技术栈

| 类别    | 选型                                 |
| ------- | ------------------------------------ |
| 构建    | Vite 5 + TypeScript                  |
| 3D 渲染 | Three.js (WebGPURenderer)           |
| 动画    | GSAP 3                               |
| 后期    | EffectComposer + UnrealBloomPass     |
| AI 配图 | MiniMax mmx-cli（image / text）      |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 类型检查 + 生产构建
npm run build

# 预览构建产物
npm run preview
```

启动后访问 `http://localhost:5173`，点击「启卷」印章按钮进入主舞台。

## 操作指南

| 操作             | 效果                                  |
| ---------------- | ------------------------------------- |
| 鼠标左键拖拽     | 旋转地球                              |
| 滚轮             | 缩放（minDistance=3.5, maxDistance=18）|
| 悬停大陆         | 显示大陆名 + HUD 高亮                 |
| 点击大陆         | 相机聚焦 + 右侧卷轴展开              |
| 点击「合卷」     | 卷轴收起 + 视角复位                   |
| 顶部视图切换器   | 大陆溯源 / 地球编年 / 列国史          |
| 空闲 2.5s        | 地球自动恢复顺时针自转                |

## 内容范围

- **大陆溯源**（8 区）：非洲、亚洲、欧洲、北美洲、南美洲、大洋洲、南极洲、北极
- **地球编年**：103 条全球时间轴，覆盖冥古宙（46 亿年前）到 2024 北极「无冰夏日」警示
- **列国史**（26 国）：中华、印度、波斯、美索不达米亚、日本、东南亚、希腊、罗马、英国、法国、俄罗斯、埃及、马里、埃塞俄比亚、津巴布韦、贝宁、玛雅、阿兹特克、印加、巴西、阿根廷、美国、加拿大、毛利、波利尼西亚、巴布亚
- **历史生物长卷**（21 张）：冥古宙 → 太古宙 → 元古·大氧化 → 雪球地球 → 寒武纪 → 泥盆纪 → 石炭纪 → 二叠纪 → 三叠纪 → 侏罗纪 → 白垩纪 → 古近纪 → 冰河时代 → 智人篝火 → 走出非洲 → 农业起源 → 金字塔 → 古罗马 → 中世纪 → 工业革命 → 现代都市

内容描述按"为什么 / 怎么发生 / 后续影响 / 小知识"四子句结构组织，约 80-150 字 / 事件。

## 内容来源

- 智人非洲起源 (Out of Africa) 与多地区演化假说
- 古人类学经典遗址：Jebel Irhoud、Zhoukoudian 周口店、Dmanisi 德马尼西、Sangiran 桑吉兰、Mungo Lake 蒙戈湖、Monte Verde 等
- 新石器农业革命：新月沃土、长江中下游、美索美洲
- 印加、毛利、波利尼西亚等文明起源
- 南极探险史与北极科考史

## 范围说明

- 仅中文界面
- 不支持 SSR
- 不引入 React / Vue（保持轻量）
- 大陆轮廓为简化多边形（聚焦于人类活动区域，不追求地理精度）
- 大气层 / 地球纹理程序化生成，无需外部资源
